## Технические решения

### Сеточный макет клавиатуры (CSS Grid)
Для ровного расположения кнопок калькулятора была использована технология **CSS Grid**, которая позволяет легко управлять колонками и отступами.

```css
/* Фрагмент из style.css */
.keyboard {
    display: grid;
    grid-template-columns: repeat(4, 1fr); /* 4 колонки равной ширины */
    gap: 10px; /* Расстояние между кнопками */
}
```
### Стилизация кнопок
Все кнопки имеют общий базовый класс .my-btn, а их цветовое различие (серые или более тёмные) реализовано через дополнительные классы .secondary и .primary.

```css
.my-btn {
    width: 60px;
    height: 60px;
    border-radius: 50%; /* Идеальный круг */
    background: rgba(255, 255, 255, 0.15); /* Эффект прозрачности */
    transition: all 0.3s ease; /* Плавность при наведении */
}
```
