## 2. Индивидуальная тематика и стилистика

В рамках выполнения требования по уникальности интерфейса была выбрана следующая концепция:

* **Тематика:** Терминал управления лунной исследовательской станцией "Sanctuary".

Интерфейс построен на базе эффекта **Glassmorphism** — элементы управления представляют собой полупрозрачные «стеклянные» панели, сквозь которые виден космический фон. Реализованы две атмосферные темы:
* **Тёмная тема (базовая):** Строгий интерфейс на фоне классической поверхности Луны.
* **Светлая тема (альтернативная):** Адаптированный интерфейс на фоне лунохода (`Moon_rover`), демонстрирующий работу на поверхности.

## 3. Теоретическая база применяемых технологий

При разработке интерактивной части были применены следующие концепции:

3. **Управление переполнением контента:** Решение проблемы длинных чисел на экране калькулятора с помощью свойств `overflow-x: auto` и скрытия стандартного скроллбара.

## 4. Архитектурные решения и фрагменты кода

### 4.1. Логика переключения тем (JavaScript)

Для смены тем написана единая универсальная функция `toggleTheme`, которая управляет состоянием всего приложения.

```javascript
// Фрагмент файла script.js
const headerThemeBtn = document.getElementById('header_theme_btn');
const moonImg = document.querySelector('.moon-bg');

function toggleTheme() {
    // Переключаем глобальный класс на body
    document.body.classList.toggle('alternate-theme');

    // Проверяем текущую тему и адаптируем контент
    if (document.body.classList.contains('alternate-theme')) {
        moonImg.src = 'Moon_rover.png';
        if (headerThemeBtn) headerThemeBtn.textContent = 'ТЕМНАЯ ТЕМА';
    } else {
        moonImg.src = 'Moon.png';
        if (headerThemeBtn) headerThemeBtn.textContent = 'СВЕТЛАЯ ТЕМА';
    }
}

if (headerThemeBtn) {
    headerThemeBtn.onclick = toggleTheme;
}
```

### 4.2. Эффект матового стекла (Glassmorphism)

Для визуального отделения элементов управления от космического фона используется размытие заднего плана.

```css
/* Фрагмент файла style.css */
.main-header, .main-footer, .calculator-panel {
    background: rgba(0, 0, 0, 0.7); /* Полупрозрачный темный фон */
    backdrop-filter: blur(10px);    /* Размытие фона под элементом */
    border: 1px solid rgba(255, 255, 255, 0.1); /* Тонкая светлая рамка */
}
```

### 4.3. Блокировка прокрутки интерфейса
Для того чтобы приложение ощущалось как нативный терминал, а не как обычный веб-сайт, была жестко зафиксирована высота экрана и отключена системная прокрутка.

```css
body {
    margin: 0;
    padding: 0;
    height: 100vh; /* Строго 100% высоты экрана */
    overflow: hidden; /* Полный запрет появления скроллбаров */
    display: flex;
    justify-content: center;
    align-items: center;
}
```
